Secure WordPress against spam comments and brute-force attacks.

== Description ==
Login With reCAPTCHA Gives you Use reCAPTCHA For Login page and It will secure from brute Froce attack.


== Installation ==
1. Upload Login With reCAPTCHA files to the "/wp-content/plugins/" directory.
2. Use the menu Tools => Login With reCAPTCHA to configure Login With reCAPTCHA.

== Frequently Asked Questions ==
= Why to install this plugin? =
It Is Easy To Use No Extra Ads or Anything its Pure code.

= How to disable this plugin? =
Just use standard Plugin overview page in WordPress admin section and deactivate it or rename plugin folder /wp-content/plugins/login-with-recaptcha over FTP access.
